How to connect Chrome Extension
  • Open Google Chrome and navigate to chrome://extensions/
  • Enable Developer Mode in the top right corner of the extension tab.
  • Click Load unpacked and select the extension directory of NexDown.
  • Once loaded, open any video page. NexDown will automatically connect to this desktop app.
